# chat-react

## About project

Local chat that works in the browser

## Getting Started

1. Clone this repository

```bash
git@github.com:Andr-Den/chat-react.git
```

2. Install dependencies

```bash
npm install
```

3. Build project

```bash
npm run build
```

4. Run project

```bash
npm run start
```

## Used technologies

1. ReactJS
